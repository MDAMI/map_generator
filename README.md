Map Generator
---

Web service to generate terrain based hex maps based on Gygax's Algorithm from AD&D

Usage
---

Start the development server with this command:

```
npm start
```

Setup
---
Front End
```
npm install
```

Back End
```
virtualenv venv
```
Activate venv depending on platform

```
pip install -r requirements.txt
```

Compile
---

```
npm run compile
```
